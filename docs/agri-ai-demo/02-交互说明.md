# 交互说明

## 1. 主流程
1. 进入页面默认展示“病害识别”场景。
2. 用户在顶部切换为“虫害检测”后，左中右所有模块同步切换。
3. 用户拖动底部时间轴，地图定位点和识别框同步更新。
4. 用户点击告警项或高风险区域，地图高亮对应地块。
5. 用户点击“打开报告”，查看并导出当前场景报告。

## 2. 联动规则
1. `scene` 变化时：重置筛选条件、选中任务、帧索引与播放状态。
2. `filter` 变化时：刷新 dashboard 与 detections 数据。
3. `selectedMissionId` 变化时：帧索引归零，时间轴重新计算总帧数。
4. `frameIndex` 变化时：刷新影像帧信息和当前识别框。
5. `selectedPlotId` 变化时：地图高亮区域、右侧列表选中状态同步。

## 3. 异常与降级
1. 接口失败：页面显示“数据加载失败”空态。
2. 无帧数据：中部显示“暂无帧数据”提示。
3. 测试环境：地图组件不初始化 WebGL，仅保持结构可渲染。

## 4. 动效说明
1. KPI 卡片使用轻量入场动画，强化实时感。
2. 报告抽屉使用右侧滑入动画。
3. 告警列表与地图热区通过颜色层级体现风险梯度。
